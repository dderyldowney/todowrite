### Installation
See detailed installation instructions: [Installation Guide](installation/INSTALLATION_GUIDE.md)

### Basic Usage
```bash
# Initialize a new ToDoWrite project
todowrite init

# Create a goal
todowrite create --layer goal --title "Build TodoWrite App" --owner "dev-team"

# List all nodes
todowrite list

# Search nodes
todowrite search "database"

# Export to YAML
todowrite export-yaml
```
